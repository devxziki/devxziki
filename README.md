## 🚀 &nbsp;Featured Project — NexusOS

<br/>

<div align="center">

<a href="https://github.com/devxziki/NEXUS-OS">
<img src="https://capsule-render.vercel.app/api?type=rect&height=4&color=0:6366f1,100:22d3ee&width=100%25" />
</a>

</div>

<br/>

> **NexusOS** is a local-first, AI-native software engineering operating system — not just a coding assistant, but a complete AI engineering team built into a VS Code-style desktop environment.

<br/>

```
NexusOS = AI IDE
        + Multi-Agent Employee System
        + Autonomous Workflow Engine
        + Browser Automation + QA Platform
        + Security Audit System
        + Architecture Generator
        + Local-First Runtime Manager
```

<br/>

### 🧠 &nbsp;How It Works

<br/>

```
User Prompt
      ↓
Nexus Manager (Orchestration Brain)
      ↓
Workflow Graph Generated
      ↓           ↓           ↓
Frontend     Backend      Security
Engineer     Engineer      Worker
      ↓           ↓           ↓
  React UI    REST APIs    ZAP + Nuclei
      ↓           ↓           ↓
         QA Engineer (Playwright)
                  ↓
            Final Review + Summary
```

<br/>

### 👥 &nbsp;AI Employee Roster

<br/>

| Employee | Role |
|---|---|
| 🧩 **Nexus Manager** | Orchestration brain — plans, assigns, reviews, summarizes |
| ⚛️ **Frontend Engineer** | React, Next.js, Tailwind, shadcn/ui — full UI execution |

### 🤖 &nbsp;Current AI Stack

<br/>

| Provider | Model | Role | Cost |
|---|---|---|---|
| **OpenAI** | GPT-4o / GPT-4.1 | Primary reasoning + code generation (via Groq API key) | Paid |
| **Groq API** | Fast inference layer | Ultra-low latency routing for OpenAI-compatible models | Paid |
| **Iris** | Gemini 2.5 Flash | Autonomous assistant — self-directed agent execution | Free |
| **Gemini CLI** | Gemini 2.5 Flash | CLI-based AI execution, free tier | Free |
| **OpenCode CLI** | MiniMax (via OpenCode) | Fully free autonomous code execution worker | Free |
| **Ollama** | Local models | *(Coming soon)* — offline-first local model support | Future |

<br/>

> **Free-first philosophy:** NexusOS runs almost entirely on free tiers — Iris (Gemini 2.5 Flash), Gemini CLI, and OpenCode CLI + MiniMax handle the bulk of work at zero cost. OpenAI via Groq covers precision-heavy workloads only when needed.

### 🗂️ &nbsp;Monorepo Structure

<br/>

```
nexusos/
├── apps/           → Electron desktop application
├── core/           → Manager, execution engine, event bus, providers
├── employees/      → 9 isolated AI employee modules
├── skills/         → Modular capability packages (React, Prisma, Docker…)
├── plugins/        → Playwright, GitHub, Docker, ZAP, Nuclei
├── runtimes/       → Bundled Node, Bun, Python, Git, browsers
├── security/       → Vulnerability scanner, pen tester, auditor
├── infrastructure/ → IPC, WebSocket, terminal (node-pty), process manager
├── storage/        → SQLite, ChromaDB, encrypted credentials, logs
└── templates/      → SaaS, fullstack, microservices, AI apps
```

### 🔐 &nbsp;Privacy-First Philosophy

<br/>

<div align="center">

| Principle | What It Means |
|---|---|
| **No Login Required** | Works like VS Code — install and go, no account needed |
| **No Cloud Sync** | All memory, logs, and code stay on your machine |
| **BYOK** | Bring your own API keys — NexusOS is not a middleman |
| **Provider Agnostic** | OpenAI + Groq, Gemini CLI, OpenCode + MiniMax — mix freely |
| **Free-First Routing** | Iris + Gemini CLI + OpenCode CLI cover most tasks for free |
| **Future Local Support** | Ollama integration planned for fully offline workflows |

</div>
